# ポートフォリオサイト

シンプルで美しいポートフォリオサイトのテンプレートです。

## 概要

このプロジェクトは、個人のポートフォリオサイトを作成するためのシンプルなHTMLテンプレートです。レスポンシブデザインに対応し、モダンなUI/UXを提供します。

## 特徴

- **レスポンシブデザイン**: すべてのデバイスサイズに対応
- **モダンなUI**: クリーンで見やすいデザイン
- **スムーススクロール**: ナビゲーションでのスムーズな移動
- **アニメーション効果**: 要素の表示時のフェードイン効果
- **お問い合わせフォーム**: 基本的なフォーム機能

## ファイル構成

```
test-mcp-server/
├── index.html      # メインHTMLファイル
├── style.css       # スタイルシート
├── script.js       # JavaScript機能
└── README.md       # このファイル
```

## 使用方法

### 1. ローカルでの実行

1. このリポジトリをクローンまたはダウンロード
```bash
git clone https://github.com/[username]/test-mcp-server.git
cd test-mcp-server
```

2. ブラウザで`index.html`を開く
   - ファイルをダブルクリック
   - またはローカルサーバーを起動して表示

### 2. ローカルサーバーでの実行（推奨）

Python 3を使用する場合:
```bash
python -m http.server 8000
```

Node.jsのhttpサーバーを使用する場合:
```bash
npx http-server
```

その後、ブラウザで `http://localhost:8000` にアクセス
